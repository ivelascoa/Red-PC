Fólder: PYTHON
Descripción: Código escrito en PYTHON 3.X para el proyecto. 

Requerimientos: 
La computadora donde se probó este código es una Mac (Mid 2012) con OSX 10.10. Se instaló PYHTON 3.4 utilizando macports, y después se instalaron los módulos relevantes al código con PIP. 

Instalación de PYTHON:

		sudo port install python3.4

Instalación de los módulos pertinentes: 

		python3.4 -m pip install pandas
		python3.4 -m pip install googlemaps

Contiene:
- roadDistances.py := Contiene un ejemplo que consulta la API de Google llamada Directions, a la cual se le consulta un par de direcciones. La forma de correrlo desde una terminal de UNIX es: 

		python3.4 roadDistances.py
